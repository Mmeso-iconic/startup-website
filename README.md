# Startup Website

This is a responsive and modern startup-themed website that showcases a tech-driven company with bold visuals, clean layout, and clear messaging. The site includes sections like a hero banner, partner logos, messaging content, and service highlights. Built using **React.js** and **CSS**, the project focuses on mobile-first design and reusable component structure for fast development and scalability.

## 📌 Introduction

This project is a **startup landing page** created to practice building professional, responsive, and visually engaging websites using React. It is component-based and styled to reflect a modern tech brand identity. The goal was to explore layout systems, prop passing, and React Router-based navigation — all while keeping performance and design responsiveness in mind.

## ✨ Features

- Mobile-first and fully responsive design
- Hero section with headline and call-to-action
- Partner logos and collaborations section
- Services/messaging section with icons and descriptions
- Sidebar menu toggle for smaller screens
- React routing for navigation
- Clean, component-based file structure
- Font styling and theming for visual consistency

---

## 🛠 Technologies Used

- **React.js** – Core JavaScript library for UI components  
- **CSS3** – Custom styling and responsiveness  
- **React Router** – Navigation and page routing  
- **Google Fonts** – "Work Sans" and "Crimson Text" for elegant typography  
- **Figma** – UI/UX reference design  
- **Vite / Create React App** – Project scaffolding (depending on setup)
